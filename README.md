# Microcontrollers

Repository dedicated to the laboratory exercises of the Microcontroller Systems class.

## Lab0

- Separation of prime numbers from a list of random numbers and sorting using **bubble sort**.

## Lab1
- Implementation of a 0-99 counter with variable step control using multiplexed 7-segment displays and binary LEDs in Assembly.

## Lab2
- Development of a memory game logic using a 16x2 LCD for visual feedback and a matrix keypad for user input.

## Lab3
- Implementation of a stepper motor winder control system using a finite state machine, a 4x4 matrix keypad and GPIO interrupts.
